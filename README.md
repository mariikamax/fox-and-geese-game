# Реализация игры Лиса и Гуси (Fox and Geese)
Классическая настольная игра с простыми правилами и стратегическим геймплеем. Реализация на Java с графическим интерфейсом Swing.

## Ссылка на видео: https://drive.google.com/file/d/1KubL1BHU8ag0QHNcjmpYbGcxJVPWiC-G/view?usp=drive_link

## Особенности
- Графический интерфейс на Swing
- Два режима игры: игрок против AI и режим наблюдателя
- Простой AI с базовой стратегией
- Проверка правильности ходов
- Автоматическое определение конца игры
- Подсветка последнего хода
- Возможность начать новую игру

## Правила игры
- `Доска: 9×9 клеток (играют только на клетках с четной суммой координат)`
- `Фигуры: 1 лиса (красная) и 13 гусей (синие)`
- `Движение гусей: только вперед (вверх) на 1 клетку по диагонали`
- `Движение лисы: в любом направлении на 1 клетку, может прыгать через гусей`
- `Победа гусей: заблокировать лису (нет валидных ходов)`
- `Победа лисы: съесть гусей, пока не останется 4 или меньше`
- `Ничья: 50 ходов без захвата`

## Режимы игры
### 1. Режим игрока (по умолчанию)
- Вы играете за лису
- AI играет за гусей
- Ходы делаются кликами мыши
- Чтобы запустить в этом режиме:
`cd C:\Users\Windows\Desktop\VSU\ООП\FoxAndGeese\target\classes`
` java ru.vsu.cs.maximova.Main`

### 2. Режим наблюдателя
- Запуск с аргументом "я наблюдатель"
- Оба игрока - AI
- Автоматическая игра с задержкой между ходами
- Чтобы запустить в этом режиме:
`cd C:\Users\Windows\Desktop\VSU\ООП\FoxAndGeese\target\classes`
` java ru.vsu.cs.maximova.Main "я наблюдатель"`
